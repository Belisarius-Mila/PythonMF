MMTX GitHub Pages

Tato slozka `docs/` je pripravena pro publikaci pres GitHub Pages.

Postup:

1. Nahraj zmeny do GitHub repozitare.
2. Otevri repozitar na GitHubu.
3. Jdi do `Settings` -> `Pages`.
4. V sekci `Build and deployment` nastav:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/docs`
5. Uloz.

Po chvili GitHub vytvori verejnou adresu webu.

Poznamky:

- Vstupni stranka je `docs/index.html`.
- Obrazek sceny je `docs/scene.jpg`.
- Hlas je reseny pres browserove `speechSynthesis`, takze kvalita hlasu se muze lisit podle prohlizece a pocitace.
